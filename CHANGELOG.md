## [1.0.0] - 2024-11-11

### Added
- Initial release of HR Suite
- One-click installation and configuration
- Automated HR Settings configuration
- Default departments creation (8 departments)
- Default designations creation (12 designations)
- Pre-configured leave types (6 types)
- Default shift type (General Shift 9 AM - 6 PM)
- Salary components (earnings and deductions)
- Custom HR roles (HR Manager Suite, HR User Suite, Employee Self Service)
- Employee automation:
  - Auto-create user accounts
  - Send welcome emails
  - Auto-allocate leaves
- Email templates:
  - Welcome Email
  - Leave Approval Notification
- Scheduled tasks:
  - Birthday reminders
  - Probation ending alerts
  - Leave balance reminders
- HR Suite Dashboard
- Complete documentation
- Installation scripts
- GitHub issue templates
