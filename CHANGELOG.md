# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-11

### Added
- Initial release of HR Suite
- One-click installation and configuration
- Automated employee onboarding
- Pre-configured departments and designations
- Default leave types (Annual, Sick, Casual, LWP, Maternity, Paternity)
- Automated leave allocation
- Email templates (Welcome, Leave Approval)
- HR Dashboard with statistics
- Employee self-service portal
- Quick actions for common HR tasks
- Scheduled tasks for birthday reminders
- Scheduled tasks for probation alerts
- Default shift configuration (9 AM - 6 PM)
- Salary components setup
- Custom roles (HR Manager Suite, HR User Suite, Employee Self Service)

### Features
- **Employee Management**
  - Auto-create user accounts
  - Send welcome emails
  - Automatic leave allocation
  
- **Leave Management**
  - 6 pre-configured leave types
  - Automated approval workflows
  - Leave balance tracking
  
- **Attendance**
  - Default shift types
  - Auto-attendance support
  - Calendar view
  
- **Payroll**
  - Pre-configured salary components
  - Automated salary slip generation
  - Email salary slips
  
- **Dashboard**
  - Real-time HR statistics
  - Quick action buttons
  - Employee metrics
  
- **Self-Service Portal**
  - Leave application
  - Attendance viewing
  - Salary slip download
  - Leave balance check

### Installation
- One-click installation script
- Complete setup script with error handling
- Automated dependency installation

### Documentation
